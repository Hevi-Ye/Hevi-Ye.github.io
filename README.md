## 如何使用

### 创建一个新的博客

```shell
~ $ gem install jekyll bundler
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ bundle exec jekyll serve
# => Now browse to http://localhost:4000
```

### 已有一个博客

```shell
~ $ cd myblog
~/myblog $ jekyll serve
# => Now browse to http://localhost:4000
```

详细文档查看[中文文档](https://www.jekyll.com.cn)，[英文文档](https://jekyllrb.com)
